# ポートフォリオサイト

## 概要
これまでの制作物や、学習した技術をまとめたWebポートフォリオサイトです。

## 使用技術
- HTML
- CSS（SCSS）
- JavaScript
- gsap

## 実装内容
- レスポンシブ対応
- ハンバーガーメニュー
- スムーズスクロール
- ファーストビューのアニメーション
- スクロールトリガーによるアニメーション
- JavaScriptによるメールアドレスの表示対策
- メールアドレスのコピー機能

## 今後の改善予定
- PHP化によるシングルページの作成
- お問い合わせフォーム実装
- TOPへ戻るボタン

## 制作期間
1週間

## 工夫した点
オレンジが好きなので自分らしさを出すため全体をオレンジにしました。見てて疲れないようトーンやグラデーションの入れ方などに気をつけ何度も調整し決めたカラーになります。

アクセントカラーも好きなブルーで統一しました。

ファーストビューのアニメーションが目に付くように意識しました。

アニメーションを導入するにつれてgsapを使い滑らかで見やすくなるよう意識しました。
