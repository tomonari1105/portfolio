# ポートフォリオサイト

## 概要
Web制作の学習の一環として制作したポートフォリオサイトです。

## 使用技術
- HTML
- CSS（SCSS）
- JavaScript
- gsap

## 実装内容
- レスポンシブ対応
- ハンバーガーメニュー
- スムーズスクロール
- ファーストビューのアニメーション
- スクロールトリガーによるアニメーション
- JavaScriptによるメールアドレスの表示対策

## 今後の改善予定
- PHP化によるシングルページの作成
- お問い合わせフォーム実装
- TOPへ戻るボタン

## 制作期間
2週間

## 工夫した点
ファーストビューのアニメーションが目に付くように意識しました。
アニメーションを導入するにつれて初めてgsapを使いましたが、JSとの違いや利点について勉強になりました。
